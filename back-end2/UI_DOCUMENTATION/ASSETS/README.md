# UI Assets & Resources

> **Design Assets, Images, and Resources for Trap-Intel Platform**

## Overview

This folder contains design assets, icons, images, and other resources referenced by the UI specifications.

## Asset Categories

### Icons & Graphics
- **Status Icons** - Visual indicators for various states
- **Navigation Icons** - Menu and action icons
- **Feature Icons** - Icons representing platform features
- **Brand Assets** - Logo variations and brand elements

### Images & Illustrations
- **Onboarding Graphics** - Welcome and setup illustrations
- **Empty States** - Placeholder graphics for empty data
- **Error Graphics** - Visual elements for error states
- **Dashboard Backgrounds** - Subtle background elements

### Design Tokens
- **Color Palettes** - Brand colors and theme variations
- **Typography Scale** - Font sizes and weights
- **Spacing System** - Consistent spacing values
- **Shadow Definitions** - Elevation and depth effects

### Mockups & Wireframes
- **Screen Mockups** - Visual representations of UI screens
- **User Flow Diagrams** - Navigation and process flows  
- **Component Variations** - Different states and configurations
- **Responsive Layouts** - Mobile and tablet adaptations

## Design Guidelines

### Color Usage
- **Primary Blue (#1a365d)** - Main brand color for primary actions
- **Security Green (#38a169)** - Success states and positive indicators  
- **Alert Red (#e53e3e)** - Errors, warnings, and critical states
- **Neutral Grays** - Background, text, and subtle UI elements

### Typography
- **Headings** - Bold, clear hierarchy for information architecture
- **Body Text** - Readable, accessible text for content
- **Code/Data** - Monospace fonts for technical information
- **Labels** - Medium weight for form labels and UI labels

### Iconography
- **Consistent Style** - Outlined icons with consistent line weight
- **Semantic Meaning** - Icons that clearly represent their function
- **Scalable Vector** - SVG format for crisp rendering at all sizes
- **Accessible** - High contrast and clear visual distinction
