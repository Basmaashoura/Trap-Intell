## Domain Reference Summary

### Aggregates & Services

| Type | Purpose | Key Properties |
|------|---------|----------------|
| `Report` | Main report aggregate | `Id`, `OrganizationId`, `Type`, `Title`, `Status`, `GeneratedAt` |
| `ReportTemplate` | Reusable report templates | `Id`, `Name`, `BaseType`, `Sections`, `IsSystemTemplate` |
| `ReportExport` | Export job tracking | `Id`, `ReportId`, `Format`, `Status`, `FileSize`, `FilePath` |
| `ReportGenerationService` | Generates reports from data | Methods: `GenerateExecutiveSummary()`, `GenerateTechnicalReport()`, `GenerateLogsExport()` |
| `ReportSchedulingService` | Manages scheduled reports | Methods: `CreateSchedule()`, `ExecuteSchedule()`, `PauseSchedule()` |
| `ReportAnalyticsService` | Analytics dashboard data | Methods: `GetSecurityScore()`, `GetKPIs()`, `GetTrendData()` |

### Key Value Objects

| Value Object | Purpose | Properties |
|--------------|---------|------------|
| `ReportPeriod` | Date range for reporting | `StartDate`, `EndDate` |
| `ReportContent` | Generated report content | `Sections`, `Charts`, `Tables`, `RawData` |
| `ReportConfiguration` | Report generation settings | `IncludedSections`, `FilterCriteria`, `OutputFormat` |
| `ScheduleConfiguration` | Schedule settings | `Frequency`, `DayOfWeek`, `DayOfMonth`, `Time`, `Timezone` |
| `DeliveryConfiguration` | Report delivery settings | `Method`, `Recipients`, `S3BucketUrl`, `WebhookUrl` |
| `SecurityPostureScore` | Overall security score | `TotalScore`, `DetectionCoverage`, `ResponseTime`, `ThreatMitigation` |
| `KeyPerformanceIndicators` | KPI metrics | `MTTA`, `MTTR`, `FalsePositiveRate`, `AlertEscalationRate` |

### Domain Events

| Event | Trigger |
|-------|--------|
| `ReportCreatedEvent` | New report created (draft) |
| `ReportGeneratedEvent` | Report content generated |
| `ReportReviewedEvent` | Report marked as reviewed |
| `ReportSentEvent` | Report sent to recipients |
| `ReportArchivedEvent` | Report archived for storage |
| `ReportTemplateCreatedEvent` | Custom template created |
| `ReportTemplateUpdatedEvent` | Template modified |
| `ReportScheduleCreatedEvent` | New scheduled report |
| `ReportScheduleExecutedEvent` | Scheduled report ran |
| `ReportSchedulePausedEvent` | Schedule paused by user |
| `ReportExportStartedEvent` | Export job started |
| `ReportExportCompletedEvent` | Export job finished |
| `ReportExportFailedEvent` | Export job failed |

### Report Generation

#### Executive Summary Sections
- Executive Overview (key highlights, trends)
- Key Metrics (attack counts, response times, uptime)
- Threat Landscape (top threats, geographic distribution)
- Risk Assessment (current risk level, recommendations)
- Incident Summary (major incidents, impact)
- AI Recommendations (automated insights)

#### Technical Report Sections
- Attack Event Analysis (detailed breakdowns, patterns)
- MITRE ATT&CK Mapping (observed techniques, tactics)
- Honeypot Performance (uptime, events captured, health)
- Threat Actor Profiles (new/active actors, analysis)
- Alert Response Metrics (MTTA, MTTR, escalation rates)
- IOC List (indicators of compromise)
- Raw Data Appendix (full event logs)

#### Logs Export Options
- Data Sources: Attack Events, Alert Logs, Honeypot Telemetry, User Activity, System Logs
- Filters: Honeypot selection, severity levels, attack types
- Formats: CSV (spreadsheet), JSON (machine readable)
- Options: Compression (ZIP), file splitting (max 50MB)

### Repository Interface

| Method | Purpose |
|--------|--------|
| `GetByIdAsync(id)` | Get single report by ID |
| `GetByOrganizationIdAsync(organizationId)` | Get reports for organization |
| `GetByTypeAsync(type)` | Get reports by type |
| `GetByStatusAsync(status)` | Get reports by status |
| `GetByCreatedByAsync(userId)` | Get reports by creator |
| `GetScheduledReportsAsync()` | Get all scheduled reports |
| `GetExportHistoryAsync()` | Get export job history |
| `SearchAsync(query)` | Search reports by title/description |

### Business Rules

| Rule | Description |
|------|-------------|
| `ReportSizeLimit` | Individual reports limited to 100MB |
| `ExportRetention` | Export files retained for 30 days |
| `ScheduleLimit` | Max 10 scheduled reports per organization |
| `TemplateLimit` | Max 25 custom templates per organization |
| `ReportRetention` | Reports archived after 2 years |
| `AccessControl` | Users can only view reports from their organization |

### Analytics & KPIs

#### Security Posture Score Components
- **Detection Coverage**: Honeypot deployment effectiveness
- **Response Time**: Average time to acknowledge/resolve alerts 
- **Threat Mitigation**: Success rate of threat response
- **System Health**: Honeypot uptime and system availability

#### Key Performance Indicators
- **MTTA**: Mean Time To Acknowledge alerts
- **MTTR**: Mean Time To Resolve incidents
- **False Positive Rate**: Percentage of false alerts
- **Alert Escalation Rate**: Percentage of escalated alerts
- **Coverage Metrics**: Assets protected, attack vectors covered
- **Threat Intelligence**: New threats identified, IOCs discovered
