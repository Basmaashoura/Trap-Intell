```
GET /api/threat-actors

  items: ThreatActor[],
  totalCount: number,
  summary: {
    severe: number,
    critical: number,

    blocked: number
  }
}
```

### Backend Reference
- `ThreatActor.Create(...)` - Factory method creates from correlation criteria
- `ThreatActor.CorrelationCriteria` - Value object with IP patterns, behaviors
- `ThreatActor.AddIP(ipAddress, geoLocation)` - Adds ThreatActorIPEntity
- `ThreatActor.AddTTP(techniqueId, tacticName)` - Adds ThreatActorTTPEntity
- `ThreatActor.AddBehaviorPattern(...)` - Adds BehaviorPatternEntity
- `ThreatActor.UpdateThreatScore(score)` - Recalculates threat score
- `ThreatActor.CalculateThreatLevel()` - Determines threat level from score
- Domain Events: `ThreatActorIdentifiedEvent`, `ThreatActorThreatLevelChangedEvent`, `ThreatActorIPBlockedByUserEvent`

## Domain Reference Summary

### Aggregate & Entities

| Type | Purpose | Key Properties |
|------|---------|----------------|
| `ThreatActor` | Aggregate root for threat actors | `Id`, `OrganizationId`, `ExternalIdentifier`, `Alias` |
| `ThreatActorIPEntity` | Associated IP addresses | `IPAddress`, `GeoLocation`, `IsBlocked`, `BlockedReason` |
| `ThreatActorTTPEntity` | MITRE ATT&CK techniques | `TechniqueId`, `TacticName`, `UsageCount`, `IsSignature` |
| `BehaviorPatternEntity` | Behavioral patterns | `PatternType`, `Description`, `OccurrenceCount`, `IsDistinctive` |
| `ThreatIntelNoteEntity` | Intelligence notes | `NoteType`, `Content`, `Source`, `IsInternal`, `IsPinned` |

### Key Value Objects

| Value Object | Purpose | Properties |
|--------------|---------|------------|
| `ThreatScore` | Threat scoring | `Value`, `AttackFrequencyScore`, `SeverityHistoryScore`, `TTPSophisticationScore`, `RecencyScore` |
| `CorrelationCriteria` | Actor correlation | `IPPatterns`, `Behaviors`, `TimeWindows` |
| `GeoLocation` | IP geolocation | `Country`, `CountryCode`, `City`, `ISP`, `ASN` |
| `MitreTechnique` | MITRE ATT&CK data | `TechniqueId`, `TechniqueName`, `TacticName` |

### Domain Policies

| Policy | Purpose |
|--------|--------|
| `ThreatActorScoringPolicy` | Calculates threat scores based on activity patterns |
| `ThreatActorCorrelationPolicy` | Correlates attacks to identify actors |
| `ThreatLevelClassificationPolicy` | Determines threat level from score |

### Domain Events

| Event | Trigger |
|-------|--------|
| `ThreatActorIdentifiedEvent` | New threat actor identified through correlation |
| `ThreatActorThreatLevelChangedEvent` | Threat level escalated/de-escalated |
| `ThreatActorIPAddedEvent` | New IP address associated |
| `ThreatActorIPBlockedByUserEvent` | IP blocked by security analyst |
| `ThreatActorIPUnblockedByUserEvent` | IP unblocked by security analyst |
| `ThreatActorTTPAddedEvent` | New MITRE technique observed |
| `ThreatActorTTPMarkedAsSignatureEvent` | Technique marked as signature |
| `ThreatActorBehaviorPatternAddedEvent` | New behavior pattern detected |
| `ThreatActorBehaviorPatternMarkedAsDistinctiveEvent` | Pattern marked as distinctive |
| `ThreatActorClassificationChangedEvent` | Actor type classification changed |
| `ThreatActorMarkedAsFalsePositiveEvent` | Actor marked as false positive |
| `ThreatActorIntelNoteAddedEvent` | Intelligence note added |
| `ThreatActorIntelNotePinnedEvent` | Note pinned/unpinned |

### Repository Interface

| Method | Purpose |
|--------|--------|
| `GetByIdAsync(id)` | Get single threat actor by ID |
| `GetByExternalIdentifierAsync(externalId)` | Get by external identifier |
| `GetByOrganizationIdAsync(organizationId)` | Get actors for organization |
| `GetActiveActorsAsync(organizationId)` | Get currently active actors |
| `GetByThreatLevelAsync(threatLevel)` | Get actors by threat level |
| `GetByIPAddressAsync(ipAddress)` | Find actors associated with IP |
| `GetByThreatActorTypeAsync(type)` | Get actors by type |
| `GetByRegionAsync(region)` | Get actors by geographic region |
| `SearchByAliasAsync(alias)` | Search actors by alias/name |

### Domain Services

| Service | Purpose |
|---------|--------|
| `ThreatActorCorrelationService` | Correlate attacks to identify threat actors |
| `ThreatActorScoringService` | Calculate and update threat scores |
| `ThreatActorClassificationService` | Classify actor types based on behavior |
| `ThreatActorGeoLocationService` | Resolve IP geolocation and ASN data |
| `ThreatActorReportingService` | Generate threat actor reports |

### Behavior Pattern Types

| Type | Purpose | Domain Value |
|------|---------|---------------|
| `TimeBased` | Temporal attack patterns | `BehaviorPatternType.TimeBased` |
| `GeographicBased` | Location-based patterns | `BehaviorPatternType.GeographicBased` |
| `ToolBased` | Tool/malware signatures | `BehaviorPatternType.ToolBased` |
| `TechniqueBased` | Attack technique patterns | `BehaviorPatternType.TechniqueBased` |
| `TargetBased` | Target selection patterns | `BehaviorPatternType.TargetBased` |
| `NetworkBased` | Network behavior patterns | `BehaviorPatternType.NetworkBased` |
| `Other` | Uncategorized patterns | `BehaviorPatternType.Other` |
